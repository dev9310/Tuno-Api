Welcome to the API

This page provides information on how to use the API and examples for testing.

API Endpoints

  • POST /api/scrapper
    Description: Fetches search results for the given query.
    Request Payload: 
    {
  "data": "your_query"
}
    Response Example: 
    {
  "status": "success",
  "data": [ ...results... ]
}
  • POST /api/artist/songs
    Description: Fetches songs and singer details for the given query.
    Request Payload: 
    {
  "query": "artist_name"
}
    Response Example: 
    {
  "status": "success",
  "data": [ ...songs... ],
  "singer": [ ...details... ]
}

How to Test

You can test the APIs using tools like Postman, Curl, or by integrating them with your frontend.

Example Curl command for /api/scrapper: 

curl -X POST https://tuno-api.vercel.app/api/scrapper \
-H "Content-Type: application/json" \
-d '{"data": "example_query"}'

Example JavaScript code for calling the /api/scrapper endpoint: 

async function callScrapperAPI() {
  const inputData = { data: "example_query" };
  const response = await fetch("https://tuno-api.vercel.app/api/scrapper", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(inputData),
  });
  const result = await response.json();
  console.log(result);
}
